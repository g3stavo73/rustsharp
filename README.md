# RustSharp

A bridge between Rust and the .NET ecosystem (ASP.NET Core + FFI).

RustSharp allows Rust code to be compiled as a native library (.so / .dll) and consumed by .NET applications and other languages via the C ABI.

---

## ⚙️ Stack

- Rust (cdylib)
- C ABI (FFI)
- .NET 9 (ASP.NET Core)
- P/Invoke (LibraryImport)
